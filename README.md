# IPL Data Analysis Project

## Introduction
IPL Data Analysis Project is a Python based project that analyzes IPL match data and provides useful insights about teams and players.

## Objective
The main objective of this project is to analyze IPL data and find:
- Most winning IPL teams
- Top batsmen by runs
- Top bowlers by wickets
- Players with most sixes
- Team win percentage

## Tools and Technologies Used
- Python
- Pandas
- Matplotlib
- Numpy
- Streamlit

## Dataset
This project uses IPL datasets:
- matches.csv
- deliveries.csv

The dataset contains information about IPL matches, teams, players, runs and wickets.

## Analysis Performed

### Team Performance Analysis
Analyzed teams based on total matches won.

### Player Performance Analysis
Analyzed player performance:
- Top run scorers
- Top wicket takers

### Sixes Analysis
Found batsmen who hit the maximum number of sixes.

### Win Percentage Analysis
Calculated the winning percentage of IPL teams.

### Streamlit Dashboard
This project also includes an interactive streamlit dashboard for IPL data analysis.
